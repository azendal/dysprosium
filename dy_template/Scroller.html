_bindDomEvents

binds the widget so it can listen to the appropiate DOM events

_buildDomStructure

builds the Dom structure for the scroller, this takes into consideration that this elements may be passed on the configuration.

_setupItemsContainerElement

Sets the right width to the itemsContainerElement so the scroller does not have to do any further computations, this also avoids the developer the need to this manually

init

Initializes the Scroller Widget

Arguments

  • config

render

Renders the widget as last child of the specified element

Arguments

scrollNext

Moves the scroller one page right if its possible to do it

Arguments

scrollNext

Moves the scroller to the item at position "index"

Arguments

  • index

scrollPrev

Moves the scroller one page left if its possible to do it

Arguments

  • items

scrollTo

Scrolls the content to the desired position

Arguments

  • x
  • y
